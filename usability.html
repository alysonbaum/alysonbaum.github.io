Usability

There are certain functions of this website that have been designed specifically for ease of use for all people.

  • The main images have alternate text should the user be unable to view them for any reason.
  • Most navigation links open within the same window, so the back button will work to return to any given page.
  • Links to outside sources such as YouTube, LinkedIn, or Google Drive (for some work samples) will open in a new tab so that the website page is not lost.
  • The video will resize itself according to the viewing window, so should the user prefer to watch the video on a portable device in landscape mode, the video will fit the window for ease of viewing.
  • All text colors are set to contrast the background so that visually impaired readers can distinguish between the text and background.
  • When hovered over, every link will become highlighted with light pink and will become underlined so that every user can easily see tell what might navigate them away from the current position.
